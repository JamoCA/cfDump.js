cfDump.js demo

Each section below shows one way to call cfDump. Popups don't auto-open - click the buttons.

1. Native call, inline, default options

cfDump(sampleObj, true, true, false)

2. Option toggles - showTypes off, inline

cfDump(sampleObj, false, true, false) - keys render without the type suffix.

3. Popup mode

cfDump(sampleObj, true, true, true) - opens a new window.

4. jQuery.dump static form

jQuery.dump(sampleObj, true, true, false) - alias of cfDump.

5. $(el).dump() plugin form

The plugin form returns the jQuery collection for chaining and discards the HTML output. Useful with popup mode; for inline rendering, call cfDump(el, ...) directly.

I am a DOM element with attributes.

6. The everything sample

One dump containing every type, so all color palettes are visible.

7. Live JSON

Paste or type JSON below and click Render. JSON only - this won't show functions, Date objects, regex, or DOM elements, since those aren't representable in JSON. Use the sections above for those.
